# CONTENIDO
-Implementación del metodo de Monte Carlo para simulaciones estocásticas y modelado probabilístico en física y finanzas cuantitativas
-Otros proyectos para acercarme a lo que se realizaría en el sector de las finanzas cuanitativas

Este repositorio contiene implementaciones de métodos de Monte Carlo aplicados a procesos estocásticos en física y modelado cuantitativo. El objetivo es comprender las simulaciones probabilísticas desde sus principios fundamentales mediante la implementación de algoritmos en Python.

Cada carpeta contiene su propioi archivo README.md.
