# CONTENIDO
Implementación del metodo de Monte Carlo para simulaciones estocásticas y modelado probabilístico en física y finanzas cuantitativas

Este repositorio contiene implementaciones de métodos de Monte Carlo aplicados a procesos estocásticos en física y modelado cuantitativo. El objetivo es comprender las simulaciones probabilísticas desde sus principios fundamentales mediante la implementación de algoritmos desde cero en Python.
